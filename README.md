# Jeff's Second Family PenPal Ministry - Frontend

The official website for Jeff's Second Family PenPal Ministry, connecting volunteers with inmates seeking meaningful correspondence.

## 🌐 Live Website
[penpalministry.org](https://penpalministry.org)

## ✨ Features
- Complete list of inmates waiting for pen pals
- Full introduction letters from Arizona inmates
- Securus and JPay system integration information
- Priority tracking for urgent cases
- Contact information for volunteer matching
